Language: java

To compile: 
	javac *.java

To run: 
	java Main -inst <INSTANCE> -alg <ALGORITHM> -time <CUTOFF TIME> -seed <SEED VALUE>

example: 
	java Main -inst Atlanta -alg LS1 -time 600 -seed 100

You will need a DATA folder inside of the code folder with the TSP files in order for the code to run.
